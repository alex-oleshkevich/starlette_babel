## Installation

Install `starlette_babel` using PIP or poetry:

```bash
pip install starlette_babel
# or
poetry add starlette_babel
```

## Features

- Locale middleware
- Multi-domain translations
- Locale selectors
- Timezone middleware
- Timezone selectors
- Locale-aware formatters
- Jinja2 integration
